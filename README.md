# Baaghi
Pakistani renegade tries to reclaim Lahore 
---
## The goal of the project
- State machine implementation
  - Enemy AI, Player detection
- Creating a decent stealth controller
- Hitman-like gameplay
- Yakuza/Like a Dragon takedowns and finishers

## Current Timeline and Roadmap
- [x] State Machine Implementation
- [x] Enemy AI and controller
- [ ] Player Stealth controller
- [ ] Take down and Finishers
- [ ] Polish
